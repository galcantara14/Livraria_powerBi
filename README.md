# 📊 Dashboard Livraria – Power BI  

![Status](https://img.shields.io/badge/status-finalizado-brightgreen)  
![Ferramenta](https://img.shields.io/badge/Power%20BI-FFB900?style=for-the-badge&logo=powerbi&logoColor=black)  

Projeto desenvolvido em **Power BI** para análise e visualização de dados de uma livraria, com foco em **vendas, estoque e clientes**.  

---

## ✨ Objetivo  

Fornecer insights sobre o desempenho da livraria através de um **dashboard interativo**, permitindo:  
- Acompanhar vendas por período.  
- Identificar os livros mais vendidos.  
- Avaliar estoque disponível.  
- Analisar perfil dos clientes.

## 🛠️ Ferramentas Utilizadas  

- **Power BI Desktop** – Criação e modelagem do dashboard.  
- **Excel/CSV** – Fonte de dados para análise.  
- **DAX (Data Analysis Expressions)** – Criação de medidas e cálculos personalizados.
